  • JSON - Saving JavaScript Array to JSON. [THIS ONE IS EXCITING!!]
  • Local Storage - Saving Task Elements and rendering them post refresh.
  • innerHtml - Appending Child Elements for each object in JSON.

Forcing myself to add even the tinest details, teaches me so much. One example was clicking on the task to cross it out. This was needed to fix bugs where only the latest task would be checked of no matter which one was pressed. The label was needed so that click the task text would also cross it out.
